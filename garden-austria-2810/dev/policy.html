SICHERHEITSPROTOKOLL

Die Site verfügt über ein SSL-Zertifikat (Secure Sockets Layer), das sicherstellt, dass sowohl die Authentizität der Website, wie die Verschlüsselung aller Informationen, die der Benutzer uns gibt. Jedes Mal, wenn sich der Benutzer registriert An die Website und liefert Informationen persönlicher Art, um ein Produkt zu kaufen, den Browser von die die Handlung ausführt, verbindet sich mit der Site über das SSL-Protokoll, das bestätigt, dass der Benutzer tatsächlich auf der Site und auf unseren Servern gefunden (was man am Erscheinen der HTTPS-Code in der Adressleiste des Browsers). Dies etabliert eine Methode zur Verschlüsselung der vom Benutzer bereitgestellten Informationen und einen eindeutigen Sitzungsschlüssel. Diese Technologie ermöglicht die personenbezogene Daten des Nutzers, wie Name, Adresse und Bankkartendaten, werden vorab verschlüsselt, sodass sie auf Reisen über das Internet nicht gelesen werden können. All die SSL-Zertifikate werden für einen bestimmten Server, in einer bestimmten Domain und für eine Entität erstellt nachweislich kommerziell.
